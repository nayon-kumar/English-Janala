## 📚 English Janala – Vocabulary Learning Web App

English Janala is a simple interactive vocabulary learning web application.  
It helps users learn English words, meanings, pronunciations, and examples.

This project uses a public API to fetch vocabulary data dynamically and display it in a clean and responsive interface.

## 🛠️ Technologies Used

- **HTML5** – Structure
- **Tailwind CSS** – Styling
- **DaisyUI** – UI components
- **JavaScript (ES6)** – Functionality
- **Fetch API** – Data fetching
- **Font Awesome** – Icons
- **Google Fonts** – Typography
- **Web Speech API** – Word pronunciation

## 🚀 Live Features

- 📖 Browse vocabulary by **lesson level**
- 🔎 **Search words** instantly
- 🔊 **Word pronunciation** using Web Speech API
- 📘 View **word details, meaning, example, and synonyms**
- ⚡ Dynamic data loading using **JavaScript Fetch API**
- 📱 Fully **responsive design**
